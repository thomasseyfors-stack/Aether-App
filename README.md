Aether: Energetic Navigation & Structural Life Mapping

Aether is a high-precision, multi-layered identity matrix calculator built for structural life analysis. 
Engineered for maximum integrity, Aether synthesizes terrestrial, soul, and theoretical energetic blueprints into a single, secure "Master Data Vault."

🛠 Structural Foundations
This application utilizes a modern reactive stack optimized for high-velocity data rendering:
Framework: React 19 (Functional Component Architecture)
Styling: Tailwind CSS (Deep Space Design Language)
Security: AES-256 encryption at rest via react-secure-storage
Disaster Recovery: Integrated Error Boundary "Blast Shields"

🌀 The Identity Matrices
Aether renders data across five distinct load-bearing layers:
Mind & Persona Mask (Tropical Placidus): Navigation for immediate terrestrial reality.
Soul & Spirit Vessel (Sidereal Lahiri): The deep-core structural foundation.
The Spark & Core Intent (Draconic): The soul’s underlying mission.
The Source & Solar Mission (Heliocentric): Sun-centered perspective of cosmic purpose.
The Theoretical Axiom (Cotsworth Plan): A 13-month calendar simulation sandbox.

🚀 Run Locally
Prerequisites
Node.js (LTS)

Setup Sequence
Install Dependencies:

Bash
npm install
Configure Environment:
Set the GEMINI_API_KEY in your secrets or a local environment file.

Initialize Sequence:

Bash
npm run dev
